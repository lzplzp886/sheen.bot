# 3. 连接到 Sheenbot Infinity

要操作 Sheenbot，首先需要通过 USB 或无线方式建立通信。

## 3.1 USB 串口连接

1. 将数据线一端插入 Sheenbot 的 USB-C 接口。  
2. 在电脑上确认驱动安装完成（Windows / macOS / Linux 均自动识别）。  
3. 打开 Mind+ 或移动端 App，选择 **串口**，然后点击 **连接**。

```plaintext
Port: COM3          # Windows
Port: /dev/ttyUSB0  # Linux
Baud: 115200
